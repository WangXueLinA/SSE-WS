## 目录结构说明

```shell
├── sse # sse实时通讯
├── ws # WebSocket实时通讯
├── openAi # 基于kimi提供的sdk搭建chatgpt
```

安装依赖
```shell
npm install
```

执行下面的命令，然后浏览器访问 http://localhost:3000

### sse实时通讯

```shell
npm run sse
```

### WebSocket实时通讯

```shell
npm run ws
```

### 基于kimi提供的sdk搭建chatgpt

```shell
npm run openAi
```
